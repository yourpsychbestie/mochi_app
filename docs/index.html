🐼 Política de Privacidad - Mochi

Última actualización: 19 de abril de 2026

1. Introducción

Bienvenido a Mochi - Tu jardín de pareja. Esta política de privacidad explica cómo recopilamos, usamos y protegemos tu información personal cuando usas nuestra aplicación.

2. Información que Recopilamos

Información de cuenta:

  • Dirección de correo electrónico
  • Nombre de usuario
  • Identificador único de usuario

Información de uso:

  • Mensajes enviados entre parejas (almacenados de forma segura)
  • Progreso en el jardín virtual
  • Respuestas a cuestionarios
  • Estadísticas de uso de la app

3. Cómo Usamos tu Información

  • Para proporcionar y mantener el servicio
  • Para sincronizar datos entre dispositivos
  • Para mejorar la experiencia del usuario
  • Para enviar notificaciones importantes (si las habilitas)

4. Almacenamiento de Datos

Tus datos se almacenan de forma segura en:

  • Firebase (Google Cloud): Para sincronización en tiempo real
  • Dispositivo local: Para funcionamiento offline

Utilizamos encriptación estándar de la industria para proteger tus datos.

5. Compartir Información

No vendemos ni compartimos tu información personal con terceros.

Solo compartimos datos cuando:

  • Es requerido por ley
  • Es necesario para proteger nuestros derechos legales
  • Tú nos das permiso explícito

6. Tus Derechos

Tienes derecho a:

  • Acceder a tu información personal
  • Corregir datos incorrectos
  • Eliminar tu cuenta y datos
  • Exportar tus datos

7. Seguridad

Implementamos medidas de seguridad para proteger tu información:

  • Conexiones encriptadas (HTTPS/TLS)
  • Autenticación segura
  • Acceso restringido a datos personales

8. Retención de Datos

Conservamos tus datos mientras mantengas una cuenta activa. Si eliminas tu cuenta, todos tus datos serán eliminados permanentemente dentro de 30 días.

9. Cambios a esta Política

Podemos actualizar esta política ocasionalmente. Te notificaremos de cambios significativos mediante la aplicación o correo electrónico.

10. Contacto

Si tienes preguntas sobre esta política de privacidad, contáctanos:

📧 Email: soporte@mochi.app

11. Consentimiento

Al usar Mochi, aceptas esta política de privacidad. Si no estás de acuerdo, por favor no uses la aplicación.

Gracias por confiar en Mochi para cultivar tu jardín de amor. 🌸
